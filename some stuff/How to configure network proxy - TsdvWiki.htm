How to set Proxy configuration

  • For browser Firefox
    • Edit -> Preferences
    • Select Network tab, click on Settings button
    • Select "Manual proxy configuration"
    • Enter: proxy.tsdv.com.vn for HTTP Proxy and default port is 3128
    • Do not need use proxy for local servers, add .tsdv.com.vn in No Proxy for text box

File:Proxy.jpg

  • Warning: Be careful, setting proxy by environment variable is not safe, you account information maybe leaked
  • For system wide 

  [root@mrbig]# vim /etc/profile
  export http_proxy=http://username:password@proxy.tsdv.com.vn:3128

  • For local user 

  [user@mrbig]# vim ~/.bashrc
  export http_proxy=http://username:password@proxy.tsdv.com.vn:3128

  • For subversion 

  [user@mrbig]# vim ~/.subversion/server
  [global]
    http-proxy-host = proxy.tsdv.com.vn
    http-proxy-port = 3128
    http-proxy-username = username
    http-proxy-password = password

  • For git (http and https protocols) 

  [user@mrbig]$ git config --global http.proxy $http_proxy
  [user@mrbig]$ git config --global https.proxy $http_proxy
  [user@mrbig]$ git config --global core.noproxy *.tsdv.com.vn

To make git remember your password: 

  # Create a new file at ~/.netrc with the following content:
  machine <servername>
  login <username>
  password <password>

  # ...and set permissions:
  chmod 600 ~/.netrc


  • For git (git protocol)
    • Install socat packet: 

  [user@mrbig]$ sudo apt-get install socat

    • Create a file /usr/bin/gitproxy with the content as bellows: 

#!/bin/sh
# Use socat to proxy git through an HTTP CONNECT firewall.
# Useful if you are trying to clone git:// from inside a company.
# Requires that the proxy allows CONNECT to port 9418.
#
# Save this file as gitproxy somewhere in your path (e.g., ~/bin) and then run
# chmod +x gitproxy
# git config --global core.gitproxy gitproxy
#
# More details at http://tinyurl.com/8xvpny

# Configuration. Common proxy ports are 3128, 8123, 8000.
_proxy=proxy.tsdv.com.vn
_proxyport=3128
_proxyuser=your_username
_proxypass=your_password

exec socat STDIO PROXY:$_proxy:$1:$2,proxyport=$_proxyport,proxyauth=$_proxyuser:$_proxypass

    • Configure git to use it 

  [user@mrbig]$ git config --global core.gitproxy gitproxy

    • Set SSL verify to false 

  [user@mrbig]$ git config --global http.sslverify false

How to configure proxy for curl

  • Open ~/.curlrc in any editor 

proxy=proxy.tsdv.com.vn:3128
proxy-user=<username>:<password>

How to configure proxy for ubuntu 10.04 (Lucid)

  • For apt-get 

  [root@mrbig]# vim /etc/apt/apt.conf
  Acquire::http::Proxy "http://username:password@proxy.tsdv.com.vn:3128/";

However, Many applications will use wget to download the data from internet. So, You should also set the proxy in /etc/wgetrc, similar with setting in bashrc file 

  [root@mrbig]# vim /etc/wgetrc
  http_proxy=...
  https_proxy=...
  ftp_proxy=...

  use_proxy = on

How to configure proxy gradle

Create a "gradle.properties" next to top "build.gradle" file with the bellowing content: 

systemProp.http.proxyHost=proxy1.tsdv.com.vn
systemProp.http.proxyPort=3128
systemProp.http.proxyUser=<your user name>
systemProp.http.proxyPassword=<your password>
systemProp.https.proxyHost=proxy1.tsdv.com.vn
systemProp.https.proxyPort=3128
systemProp.https.proxyUser=<your user name>
systemProp.https.proxyPassword=<your password>

How to configure proxy for windows 2010 and above (client or Server)

  • Setup proxy in IE
  • Application like firefox, thunderbird, netbeans...: just select "use system network setting"
  • With system apps 

  1. Type 'cmd' in search/menu screen
  2. Right click on the 'cmd' icon
  3. Select 'Run as Addministrator'

On the cmd screen, type the bellow code: 

netsh winhttp import proxy ie
