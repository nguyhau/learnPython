2- Clone git
This command will clone source code from repository: 

git clone <url>


3- Commit git
Commit your change to local repository. Caution: After this command, there is no actually data send to git server 

git commit [file/folder] -m <your commit message>

For easier to manage files/folders that you want to commit in Linux, you should use Meld software, using this command in terminal : 

meld .

to show all changes of your files/folders, then right click and select 'commit' files/folders.

4- Push git
If you want to make your change synchronize with source code on server, you need push command 

git push origin <branch name>

Here, master is the default branch name.
Caution: Because maybe there is many users can commit/clone/push at the same time, your source code may not up to date. At that time, you need: 

git pull     # To let git merge server source code with your local change
git push origin <branch name>

Caution: When you run git pull, conflict may occurs, at this time, please: 

git status 
# To see which files have been merged, and which ones need your attention. 
# Look for ====== conflict markers in each unmerged file, edit the surrounding text to fix the problem
git add <filename> # to mark that file as resolved
git commit # to complete the merge
# After that, you can push as normal

Caution: When you run git push, maybe other persons already push some commits to server and you can not push, please do rebase.

5- Rebase and merge Example: source code on Server has revision is C2, then, user uA and uB clone to them PC. uB commit and push C3, C4 to server. At this time, uA still has source code at revision C2. He commit C5, but cannot push to server. To make uA modification apply to server, he can rebase: 

# Rebase current modification in current branch to branch..<remote>
git rebase

If uA created a branch B, and C5 is in B, he can merge his work to master (B and master are different branches) by 

# When current branch is B
git rebase master

# When current branch is not B
git rebase master B
